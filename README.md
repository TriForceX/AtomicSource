# Atomic Source
Mod for Jedi Knight II: Jedi Outcast Multiplayer By Tox Laximus

# Changelog
A summary all changes made after Tox release from 02-26-2026:
- Merge Atomic source code into latest 1.04 code-base from 2013 
- Add bot safe path check (prevent from falling or stuck on walls)
- Add player safe teleport command (prevent stuck on solids)
- Fix missing prototypes (wrong or bad functions declarations)
- Fix jetpack code (use entity flags only and tweak general checks)
- Add back noclip feature (restore player movement state)
- Add new message for all force powers on player menu
- Fix spectator behavior (proper player cycle and stop following)
- Fix saber toggle spam using jetpack
